# Краткая сводка проекта

## Выполненная работа

Разработана полная база данных для системы управления продукцией мебельной компании.

## Основные результаты

### 1. База данных в 3НФ
- 5 таблиц с правильными связями
- Первичные и внешние ключи
- Ссылочная целостность
- Индексы для оптимизации

### 2. Структура БД

**Справочные таблицы:**
- Material_types (4 записи) - типы материалов
- Product_types (6 записей) - типы продукции  
- Workshops (12 записей) - цеха

**Основная таблица:**
- Products (20 записей) - продукция

**Связующая таблица:**
- Product_workshops (130 записей) - связь продукции с цехами

### 3. Импорт данных
Все данные из Excel файлов успешно импортированы:
- Material_type_import.xlsx ✓
- Product_type_import.xlsx ✓
- Workshops_import.xlsx ✓
- Products_import.xlsx ✓
- Product_workshops_import.xlsx ✓

### 4. ER-диаграмма
Создана в двух форматах:
- HTML (легко конвертируется в PDF через браузер)
- PlantUML (для профессиональной генерации)

### 5. Функционал
Реализованы все требуемые функции:
- ✅ Просмотр списка продукции
- ✅ Добавление/редактирование данных о продукции
- ✅ Просмотр списка цехов для производства

## Файлы для сдачи

1. **full_database_script.sql** - полный скрипт БД
2. **ER_diagram.pdf** - ER-диаграмма (создать из ER_diagram.html)
3. **README_DATABASE.md** - документация (опционально)

## Как создать PDF диаграммы

Самый простой способ:
1. Откройте `ER_diagram.html` в браузере
2. Ctrl+P → Сохранить как PDF
3. Готово!
