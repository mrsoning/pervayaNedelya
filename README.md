# Система управления базой данных мебельной компании

## Описание проекта

Данная система представляет собой веб-приложение для управления базой данных мебельной компании. Приложение разработано с использованием Flask и SQLite для обеспечения эффективного управления производственными данными.

## Основные возможности

- Управление каталогом продукции
- Контроль производственных цехов
- Аналитические отчеты и статистика
- Поиск и фильтрация данных
- Добавление, редактирование и удаление записей

## Технические характеристики

- **Backend**: Python Flask
- **База данных**: SQLite
- **Frontend**: HTML, CSS, JavaScript
- **Архитектура**: MVC

## Структура базы данных

Система включает следующие основные таблицы:
- Products (Продукция)
- Workshops (Цеха)
- Product_types (Типы продукции)
- Material_types (Типы материалов)
- Product_workshops (Связи продукция-цеха)

## Запуск приложения

```bash
python web_app.py
```

Приложение будет доступно по адресу: http://127.0.0.1:5000
